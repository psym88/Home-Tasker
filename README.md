# Home Tasker has moved

> [!IMPORTANT]
> This repository is archived and no longer maintained.

Due to extensive refactoring, active development continues under the new name **Tasks** in the [`psym88/ha_tasks`](https://github.com/psym88/ha_tasks) repository.

[![Open your Home Assistant instance and add the new repository to HACS](https://my.home-assistant.io/badges/hacs_repository.svg)](https://my.home-assistant.io/redirect/hacs_repository/?owner=psym88&repository=ha_tasks&category=integration)

Please use the new repository for installation, releases, documentation, issues, and feature requests:

- [Tasks repository](https://github.com/psym88/ha_tasks)
- [Latest release](https://github.com/psym88/ha_tasks/releases/latest)
- [Issues and feature requests](https://github.com/psym88/ha_tasks/issues)

## Migration notice

Tasks uses the new integration domain `tasks` together with new entity, event, API, and storage paths. Existing Home Tasker data and backups are not migrated automatically.
